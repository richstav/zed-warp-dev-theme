# VS Code - Warp Dark Theme

A dark theme for Visual Studio Code.

## Installation

### Manual
```bash
mkdir -p ~/.vscode/extensions/warp-dark
cp warp-dark-color-theme.json ~/.vscode/extensions/warp-dark/
```

### Via VS Code
1. Copy theme file to extensions folder
2. Restart VS Code
3. Select "Warp Dark" from color theme

## Files

- `warp-dark-color-theme.json` - Color theme
- `settings.json` - Recommended editor settings
