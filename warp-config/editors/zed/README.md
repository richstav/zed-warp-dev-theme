# Zed Editor - Warp Dark Theme

A dark theme for Zed editor inspired by Warp.dev terminal.

## Installation

### Manual
```bash
cp -r . ~/.local/zed/extensions/warp-dark
```

### Via Zed
1. Open Zed
2. Go to Extensions (Cmd+Shift+X / Ctrl+Shift+X)
3. Search for "Warp Dark"
4. Install

## Files

- `extension.toml` - Extension manifest
- `themes/warp-dark.json` - Theme definition

## Theme Settings

The theme includes:
- Deep black background (#000000)
- Cyan accent (#00c2ff)
- Neon green syntax highlighting
- Full syntax highlighting support
